What is CSS?
- Cascading Style Sheets

Cascading
- How to style an element when several rules apply

Style
- The various styles that can be applied to HTML elements

Sheets
- Their own documents which can be linked to HTML fiels

CSS is used to control the presentation and layout of HTML

CSS box model
- CSS views every HTML element as if it has an invisible box around it
- All HTML elements are by default either a block ot inline element

Block Elements
- Starts on a new line
- Takes up full width of a page
- <h1> <p> <ul> <li>

Inline Elements
- Does not start on a new line
- Only occupies width of their content
- <a> <img> <br> <input>

CSS Properties

Content
- Inside the box
    - font size
    - font weight
    - background color

Box
- The box itself
    - width & height
    - border
    - margin

Layout
- Where boxes go
    - flexbox
    - grid
    - position


CSS Rules

h1{
    font-size: 20px;
    color: blue;
}
