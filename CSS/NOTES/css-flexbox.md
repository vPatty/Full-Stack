What is flexbox?
- Flexbox is a one-dimensional layout model making it simple to arrange items in rows or columns and distribute space

Flexbox has 2 main components
1. Flex Container (parent)
2. Flex Items (children) - this is the content inside the container


Main and Cross axis
- The main acis is the primary direction in which flex items are laid out & cross axis determines distribution on secondary axis

Flex Direction
- Controls orientation of the main axis
4 possible values
1. flex-direction: row;(default, right to left)
2. flex-direction: row-reverse;(flips the row, left to right)
3. flex-direction: column;(arranges on column, top to bottom)
4. flex-direction: column-reverse;(flips column, bottom to top)
