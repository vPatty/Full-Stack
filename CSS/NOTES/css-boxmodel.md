Default Block-Level Element Dimensions
- By default block-level elements are just big enough to fit its contents vertically and stretches full-width horizontally

The default box dimensions can be overwritten by the width and heigh properties

Block Level sizing guidance 
- Usually do not set heigh but let it be automatically determined
- Usually do not set widht which will stretch to fill the parent container
