Horizontal Centering A Container

Box Model
- Margin can be applied left and right to horizontally center child containers
margin: 0 auto; (0 top and bottom, auto left and right)

Flexbox
- Flex properties make it simple to horizontally center flex items
justify-content: center;
align-items: center; 

Horizontal Centering A Container
 
Box Model
- There is no easey way to vertically center a child container inside a parent container with a set height


Horizontal Centering Guidance
- Margin is often used to establish a webpages main container as it is a simple method for horizontal centering

- Flex box makes it simple to horizontally center regular content containers

- Padding is used to create equal internal space left and right around containers like cards and elements like anchor tags 


Vertical Centering Guidance
- Flexbobx properties are often used to vertically center UI elemnts inside a flex containers that has a set height

- Padding is used to create equal internals space top and bottom around contianers like cards and elements like anchor tags

- Padding can also be used to create equal internal space top and bottom on flex containers
