CSS Grid

What is CSS Grid?
- CSS Grid is a Two-Dimensional layout model makign it simple to arrange items in rows and columns

CSS Grid has 2 main components similar to flexbox
1. Grid Container(Parent)
2. Grid Items(Children)

CSS Grid has 2 axis
1. Column axis
2. Row axis

YOU CANNOT CHANGE THE DIRECTION OF THE AXIS

The grid is divided into grid cells by grid lines

Grid lines define grid tracks (rows and columns)
Grid rows and columns can have spacing between them

The display property on a container element will activate grid layout features on children elements
display: grid;

CSS Grid can be used for complex two-dimensional layouts
Grid is also commonly used for major sections
Grid is also commonly used for UI elements with consistent spacing in a one-dimensional row or column
