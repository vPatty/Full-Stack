HTML is the foundation of a webpage!!

It is needed on every webpage in order to display content


HTML TAG
- A tag wraps content and provdes instructions to the browser

Paragraph tag <p> </p> defines a paragraph, then entire thing makes up an HTML ELEMENT

HTML Document Structure

- Every HTML document requires 'Biolerplate' code
    - Boilerplate code refers to sections of code that are used repeatedly in many different programs or parts of the same program with little or no change

DOC STRUCTURE

<!DOCTYPE html> -- Flags to the browser that it is an HTML5 file, this is NOT AN HTML ELEMENT but very important
<html> -- The root HTML element which wraps all other elements
    <head> -- Contains key information about the document that does not appear on the webpage
        <title>I am a title</title>
    </head>
    <body>
        <p>I am a paragraph</p>
    </body>
</html>
