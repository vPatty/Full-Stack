List Types
There are 2 list types in HTML that are used for different purposes
1. Ordered
2. Unordered




1) ORDERED LIST
- A list with numbers that presents items in a clear and organized way

Ordered List Tags
<ol>
    <li></li>
</ol>


2) UNORDERED LIST
- A list with bullet points to present items in an easy to read way

Unordered List Tags
<ul>
    <li></li>
</ul>
