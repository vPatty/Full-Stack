3 ways to create a string

1. '...' - use by default, (easier to read and type)
2. "..." - use if there is a single quote inside the string
3. `...` - Template Strings
4. Escape Character
    - Character
    1. Letter (a,b,c)
    2. Number (1,2,3)
    3. Symbol (!, @, #)
    4. Escape Character (\', \", \n = new line) ex: ('I\'m Learning JavaScript)
